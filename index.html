သင်ခန်းစာများ

ကွန်ပျူတာ အခြေခံ သဘောတရားများ

အောက်ပါစာရင်းသည် ကွန်ပျူတာအခြေခံသဘောတရားများကို လေ့လာရာတွင် အထောက်အကူဖြစ်စေရန် ရည်ရွယ်ပါသည်။

၁။ ကွန်ပျူတာ အစိတ်အပိုင်းများ

  • Hardware (CPU, RAM, Hard Drive, Motherboard, GPU, etc.)
  • Software (Operating System, Applications)
  • Input Devices (Keyboard, Mouse, Microphone, Scanner)
  • Output Devices (Monitor, Printer, Speakers)

၂။ Operating System (OS)

  • Windows, macOS, Linux
  • Desktop, Icons, Folders, Files
  • Taskbar, Start Menu
  • Recycle Bin

၃။ အင်တာနက် အသုံးပြုခြင်း

  • Internet, World Wide Web (WWW)
  • Websites, Web Pages, URL
  • Web Browsers (Chrome, Firefox, Safari)
  • Links, Bookmarks, Tabs
  • Download, Upload

၄။ အီးမေးလ် (Email)

  • Email Address
  • Sending and Receiving Emails
  • Spam, Phishing

၅။ ကွန်ပျူတာ လုံခြုံရေး

  • Antivirus Software
  • Firewall
  • Malware, Spyware, Ransomware
  • Strong Passwords
  • Two-Factor Authentication (2FA)
  • VPN (Virtual Private Network)

၆။ ဖိုင် စီမံခန့်ခွဲခြင်း

  • Save, Copy, Paste, Delete, Rename
  • Search, Zoom, Scroll
  • Click, Double-Click, Right-Click
  • File Extensions

၇။ အခြေခံ ကွန်ပျူတာ အသုံးအနှုန်းများ

  • Bit, Byte
  • Driver
  • Plug and Play
  • Troubleshooting
  • Default Settings
  • Compatible
  • Upgrade, Update, Restore, Backup, Recovery

၈။ ဆော့ဖ်ဝဲ ဖွံ့ဖြိုးတိုးတက်ရေး အခြေခံများ

  • Open Source Software
  • Proprietary Software
  • Algorithm
  • Programming Language
  • HTML, CSS, JavaScript
  • Database
  • Server, Client

အထူးမှတ်ချက်: ဤစာရင်းသည် ကွန်ပျူတာအခြေခံသဘောတရားများကို လေ့လာရန်အတွက် အထောက်အကူဖြစ်စေရန် ရည်ရွယ်ပြီး အသေးစိတ်အချက်အလက်များ ပါဝင်နိုင်ပါသည်။

ကွန်ပျူတာ အခြေခံ စမ်းသပ်မှုများ

အောက်ပါအပိုင်းများမှ စမ်းသပ်မှုတစ်ခုကို ရွေးချယ်ပါ

စမ်းသပ်မှုအပိုင်းတစ်ခုကို ရွေးချယ်ပါ။ စမ်းသပ်မှုတစ်ခုစီတွင် မေးခွန်း ၁၀ ခုပါဝင်ပါသည်။

စမ်းသပ်မှုပြီးဆုံးပါက Digital Certificate ရရှိမည်ဖြစ်ပါသည်။
